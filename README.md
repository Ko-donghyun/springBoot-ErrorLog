## 개발 환경 설정
 - mac OS
   - xCode 설치
     - App Store
   - brew 설치
     - [Homebrew](http://brew.sh/) 참고
   - Java 설치
     - `brew cask install java`
   - Maven 설치
     - `brew install maven`

## 목적
 - Spring Boot에 대해서 익숙해 질 수 있도록

 - Spring Boot를 이용한 Rest 서비스
 - iText를 이용한 PDF 파일 생성
 - 에러 로그 처리
  - Exception Error
  - Garbage Collection Log
